# iview定制主题：变量覆盖方式及报错解决

# 使用方式

参考：定制主题 -- iView

  1. 安装 less 及 less-loader

    npm install less less-loader --save-dev
    1

  2. 在项目下新建一个 less 文件用来覆盖iView的变量。 less文件地址按需求存放即可。

    @import '~iview/src/styles/index.less';
// Here are the variables to cover, such as:
@primary-color: #8c0776;
    1
    2
    3

  3. 在入口文件 main.js 内导入这个 less 文件

    import Vue from 'vue';
import iView from 'iview';
import 'iview/dist/styles/iview.css'
import '../my-theme/index.less'; //根据文件地址填写

Vue.use(iView);
    1
    2
    3
    4
    5
    6

# 报错及解决方法

  1. This relative module was not found 解决方法：找到 '/build/webpack.base.config.js' 文件，删除 less-loader 相关配置，例如：

    {
    test: /\.less$/,
    loader: "style-loader!css-loader!less-loader",
},
    1
    2
    3
    4

  2. Module build failed: inline javaScript is not enabled. 解决方法：找到 '/build/utils.js' 文件中的 generateLoaders 函数，在如图所示位置加入 javascriptEnabled: true。
